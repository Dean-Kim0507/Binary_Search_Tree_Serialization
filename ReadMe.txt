1. Word Tracker Binary Search Trees and Serialization

2. What the program does: Read all of the words in the text file and store the Binary Tree (Completed) and then
 print by option:
 -f: store the textfile to physical binary file
 -pf: print in alphabetic order all words along with the corresponding list of files in which the words occur.
 -pl: print in alphabetic order all words along with the corresponding list of files and numbers of the lines in which the word occur
 -po: print in alphabetic order all words along with the corresponding list of files, numbers of the lines in which the word occur and the frequency of
      occurrence of the words.
